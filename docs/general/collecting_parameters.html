Collecting Parameters

We might have a method that accumulates the results of a sequence of operations inside a variable and returns it.

class ScienceExperiment
  def collect_results
    results = []
    results << :result_a
    results << :result_b
    results << :result_c
    results
  end
end

This is okay for simple methods, but if we're collecting the results of more complex operations, we may wish to employ a Collecting Parameter.

class ScienceExperimentWithCollectingParameter
  def collect_results
    results = []
    perform_complex_computation_a(results)
    perform_complex_computation_b(results)
    perform_complex_computation_c(results)
    results
  end

  def perform_complex_computation_a(results)
    results << :result_a
  end

  def perform_complex_computation_b(results)
    results << :result_b
  end

  def perform_complex_computation_c(results)
    results << :result_c
  end
end
